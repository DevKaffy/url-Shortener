# Url Shortener
Copy long links from any browser, shorten it in my project, copy the shortened links to a new browser and get thesame result.

## Prequisite
- Your browser version should be up to date for better experience

## Features
- Home page

## Tech/framework used
- Reactjs
- Javascript
- TailwindCss
